🏦 Banking Customer Behavior Analysis using Python, PostgreSQL & Power BI

📊 Banking Customer Behavior Analysis

🧾 Overview
This project demonstrates a complete end-to-end data analytics workflow, transforming raw banking data into actionable insights.
The analysis focuses on customer financial behavior, uncovering patterns in income, savings, loans, and account usage to support business decision-making.

📁 Dataset
Total Records: ~3,900
Features: Financial + demographic attributes

🔍 Data Includes:
Customer demographics (Age, Gender)
Financial metrics:
Estimated Income
Bank Deposits
Bank Loans
Credit Card Balance
Savings & Checking Accounts
Superannuation Savings
Foreign Currency Accounts

🧹 Data Cleaning:
Handled missing values
Removed inconsistencies
Feature engineering:
Income Bands
Customer Segments
